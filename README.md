# OutPlay
一款发布及查阅户外解析运动的app.

应用特点:
1. 用户可在首页查阅最新的文章,并对详细页面中对其收藏,评论.
2. 可将二手商品页发布或查阅心仪商品.
3. 接入百度地图API,可在应用中直接查询地点位置.
4. 接入Bmob后端云服务,可在忘记密码时通过绑定手机号找回.
5. 区别于常见的侧边栏菜单,以圆形菜单作为不同功能项的入口.
